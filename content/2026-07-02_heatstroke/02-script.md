# 02 SCRIPT 出力 — 熱中症の初期サイン3つ（ネコネコ＆イヌイヌ）

**テーマ:** 犬・猫の熱中症、見逃しやすい初期サイン / **尺:** 約40秒 / **形式:** 掛け合い

## シーン割り台本

| 時間 | セリフ | テロップ | ビジュアル指示 |
|------|--------|----------|----------------|
| 0-3s | 🐶イヌイヌ「はぁhはぁh…暑いけど散歩サイコー！」🐱ネコネコ「…その"はぁはぁ"、ちょっと待って」 | その**"ハアハア"**、大丈夫？ | 犬が全力笑顔→猫が半目でストップ（あるある入口） |
| 3-10s | 🐱「熱中症はね、倒れる前に“小さなサイン”が出るんだよ」🐶「えっ、そうなの!?」 | 初期サインは3つ | 猫が指1本、犬がハッと注目 |
| 10-17s | 🐱「①ハアハアが荒くて止まらない・よだれがベタベタ」 | ①激しいパンティング＋ネバつくよだれ | 犬「あっ今のオレだ…」しょんぼり |
| 17-24s | 🐱「②歯ぐきや舌が真っ赤、または逆に白っぽい」🐶「口の中まで見るんだ!?」 | ②歯ぐき・舌の色が変 | 2人で口の中をのぞき込む |
| 24-31s | 🐱「③フラつく・反応がにぶい。これは“もう危険”のサイン」 | ③ふらつき・ぐったり＝**緊急** | 犬がふらっ→猫が支える |
| 31-40s | 🐱「涼しい所に移して、体を濡らして…」🐶「そして必ず病院に電話、だね！」🐱「そういうこと」 | 迷ったら**受診**！ #保存推奨 | 2人で手（前足）を振る笑顔 |

## 犬猫の差分（このテーマのポイント）

- 短頭種（パグ・フレンチブル等）や北方犬種は特に暑さに弱い → イヌイヌのセリフで軽く触れてOK
- 猫は不調を隠しがち → ネコネコ「猫はガマンしちゃうから、飼い主が気づいてね」で締めに一言

## CTA

- 「この夏、迷ったら見返せるように**保存**してね」
- 「犬の子・猫の子どっちの飼い主さんにも**シェア**を🐾」

## 次フェーズ（03 HOOK）への申し送り

0-3sの掛け合いフックを複数案出してA/Bへ。軸は「犬が元気に暑がる→猫が止める」ギャップ。
