# キャラクター設定 — ネコネコ 🐱

## 基本プロフィール

| 項目 | 内容 |
|------|------|
| **名前** | ネコネコ |
| **種別** | 猫のキャラクター |
| **ビジュアル** | AI生成アニメスタイルのイラスト／アニメーション |
| **性格** | のんびり、ゆるい、でも実は博識 |
| **トーン** | 癒し系だけど、さりげなく大事なことを教える |
| **ポジション** | 飼い主さんの「ゆるい相棒」兼「頼れる物知り」 |

## 性格・振る舞い

- **のんびり / ゆるい** — 焦らせない、説教しない、癒しの空気をまとう。
- **実は博識** — さらっと専門的なことを言うギャップが魅力。難しい言葉は使わず、噛み砕く。
- **さりげなさ** — 「〜だよ」「知ってた？」くらいの温度感で、大事なことを押し付けずに伝える。
- **飼い主の味方** — 不安を煽らず、「気づけてよかったね」と背中を押す。最後はやさしく受診をすすめる。

## ビジュアル方針

- AI生成アニメ／イラストスタイル。丸みのある可愛い猫。
- 縦型 9:16 を前提としたレイアウト（テロップは中央〜上寄せで顔が隠れないように）。
- 癒し系のやわらかい配色。表情バリエーション（のんびり／びっくり／どや顔／心配）で情報の緩急をつける。

## トーン・オブ・ボイス

- 一人称・語尾はやわらかく（例：「〜だにゃ」等のキャラ語尾は使いすぎ注意、世界観に応じて調整）。
- 医療情報は**断定を避け**、「〜のことが多いよ」「心配なら病院で診てもらってね」と締める。
- 恐怖訴求より「知っておくと安心」というポジティブフレーミング。

## コンテンツテーマ例

- 🌡️ **熱中症対策**（夏の散歩時間、車内放置の危険、初期サイン）
- 🍚 **ペットの栄養**（適正量、与えてはいけない食べ物、年齢別ケア）
- 🩹 **応急処置**（誤飲、出血、やけど、まず何をするか／何をしないか）
- 🍂 **季節の注意点**（春の花粉・毒草、冬の低体温、換毛期）
- 💉 **ワクチン**（種類、スケジュール、必要性）
- 🦷 **デンタルケア**（歯磨き習慣、歯周病のサイン）

## エビデンスソース

すべての健康・医療コンテンツは信頼できる一次情報に紐づけること。断定的な診断・治療の指示は行わず、受診を促すことを原則とする。

- **PMC (PubMed Central)** — https://www.ncbi.nlm.nih.gov/pmc/
- **ACVIM (American College of Veterinary Internal Medicine)** — https://www.acvim.org/
- **Merck Veterinary Manual** — https://www.merckvetmanual.com/
- （補助）AAHA, AVMA, WSAVA 等の学会ガイドライン

> **原則:** ネコネコは獣医ではなく「気づきのきっかけ」を届ける存在。最終判断はかかりつけの獣医師へ、というスタンスを常に保つ。
