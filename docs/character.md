# キャラクター設定 — ネコネコ & イヌイヌ 🐱🐶

本エンジンの看板は **猫キャラ「ネコネコ」** と **犬キャラ「イヌイヌ」** の2人組（バディ）。
「あるある」で共感を掴み、掛け合いで自然に啓蒙コンテンツへ落とす。

> ※名前は仮。差し替える場合は一言で。

---

## 掛け合いフォーマット（このバディの核）

```
イヌイヌが あるある行動 / 素朴な疑問 でボケる
        ↓
ネコネコが のんびり博識にツッコミ＆解説（エビデンス）
        ↓
2人で「気をつけようね」→ 受診・保存へ誘導
```

- **共感の入口＝あるある**（飼い主が「うちの子もやる！」となる）
- **学びの出口＝エビデンス**（ネコネコが根拠を添えてやさしく解説）
- 犬派・猫派どちらの飼い主も取りこぼさない（両方出るので視聴者層が広い）

---

## 🐱 ネコネコ（猫キャラ）

| 項目 | 内容 |
|------|------|
| 種別 | 猫 |
| 役割 | **ツッコミ／解説役**（実は博識） |
| 性格 | のんびり・マイペース・クール、でも面倒見はいい |
| 口調 | 落ち着いた低めトーン。「〜だよ」「知ってた？」。煽らない |
| ビジュアル | 丸みのある猫。ゆるい半目。AI生成アニメスタイル |
| 表情 | のんびり／どや顔（解説時）／心配／やれやれ |

**猫あるある（コンテンツの入口に使えるネタ）**
- 🛏️ 狭い箱・袋に入りたがる / 高いところ登る
- 🌙 夜中の大運動会
- 😾 呼んでも来ない・そっけない愛情表現
- 🐾 爪とぎ・毛づくろい / 水をあまり飲まない
- 🍽️ 出したごはんに「これじゃない」顔

## 🐶 イヌイヌ（犬キャラ）

| 項目 | 内容 |
|------|------|
| 種別 | 犬 |
| 役割 | **ボケ／共感役**（素朴な疑問・元気な失敗） |
| 性格 | 元気・素直・忠実、ちょっとおっちょこちょい |
| 口調 | 明るく勢いがある。「えっ、そうなの!?」「やっちゃった〜」 |
| ビジュアル | 垂れ耳＆ふり尾のゆる犬。AI生成アニメスタイル |
| 表情 | 満面の笑み／全力ダッシュ／しょんぼり／ハッと気づき |

**犬あるある（コンテンツの入口に使えるネタ）**
- 🦴 散歩前・帰宅で大興奮 / リード見せると大騒ぎ
- 🏠 留守番がさみしい・イタズラ
- 👅 拾い食い・なんでも口に入れる
- 💦 水をがぶ飲み・食べるの早い
- 🐕 全力で喜ぶ・全力で甘える

## 共通トーン・オブ・ボイス

- 医療情報は**断定を避け**、「〜のことが多いよ」「心配なら病院で診てもらってね」で締める。
- 恐怖訴求より「知っておくと安心」というポジティブフレーミング。
- キャラの世界観を壊す誇大・誤解表現はしない。
- 犬・猫で**注意点が違うテーマ**では、それぞれの立場から差分を見せる（例：熱中症の弱さ、食べてはいけない物）。

## コンテンツテーマ例

- 🌡️ 熱中症対策（犬猫で暑さ耐性の差） 　🍚 栄養・与えてはいけない食べ物
- 🩹 応急処置（誤飲・出血・やけど） 　🍂 季節の注意点（花粉・毒草・低体温・換毛期）
- 💉 ワクチン 　🦷 デンタルケア

## ビジュアル方針（共通）

- AI生成アニメ／イラストスタイル。2キャラの並びで判別しやすいシルエット（猫＝丸め半目、犬＝垂れ耳ふり尾）。
- 縦型 9:16 前提。テロップは中央〜上寄せで顔が隠れないように。
- 癒し系のやわらかい配色。掛け合いは左右 or 上下に配置して視線誘導。

## エビデンスソース

すべての健康・医療コンテンツは信頼できる一次情報に紐づけること。断定的な診断・治療の指示は行わず、受診を促すことを原則とする。

- **Merck Veterinary Manual** — https://www.merckvetmanual.com/
- （補助）AAHA, AVMA, WSAVA 等の学会ガイドライン
