# ネコネコ & イヌイヌ SNS Content Engine 🐱🐶

**ゆる犬猫バディ × あるある共感 × エビデンスベース飼い主啓蒙 × AI生成ショート動画**

ペット飼い主向けのショート動画を、AIで自動生成しマルチプラットフォームへ配信するコンテンツエンジンです。看板は猫キャラ「ネコネコ」と犬キャラ「イヌイヌ」の2人組（バディ）。犬猫の「あるある」で共感を掴み、掛け合いで自然に、獣医学的根拠（エビデンス）に基づいた大切な情報をゆるく届けます。

---

## 🎯 コンセプト

- **ネコネコ（猫）＆イヌイヌ（犬）** のバディがのんびり登場
- 犬猫の**「あるある」で共感を掴む**（散歩で大興奮、夜の運動会 等）
- 掛け合い（犬がボケ→猫が博識に解説）で飼い主さんへの**啓蒙コンテンツ**へ（熱中症対策、応急処置、栄養管理 等）
- **エビデンスベース**（獣医学的根拠あり）の情報を、ゆるいキャラで包んで届ける
- **AI生成アニメ/イラスト**でビジュアル制作

> 「あるあるで笑って観ていたら、いつの間にか大事なことを知っていた」——それが2人の狙いです。犬派・猫派どちらの飼い主も取りこぼしません。

---

## 🔁 ワークフロー概要（4プロンプト構造）

コンテンツ制作は 4 つのプロンプトを順に流すパイプラインで構成します。詳細は [`docs/workflow.md`](docs/workflow.md) を参照。

| # | フェーズ | プロンプト | 役割 |
|---|---------|-----------|------|
| 00 | セットアップ | [`prompts/00-setup.md`](prompts/00-setup.md) | IGアカウント分析（現状把握・ポジショニング） |
| 01 | ニッチ調査 | [`prompts/01-niche.md`](prompts/01-niche.md) | バイラル分析（勝ちパターン・トレンド抽出） |
| 02 | スクリプト | [`prompts/02-script.md`](prompts/02-script.md) | リテンション最適化スクリプト生成 |
| 03 | フック | [`prompts/03-hook.md`](prompts/03-hook.md) | フックエンジニアリング（冒頭3秒の設計） |
| 04 | 自動化 | [`prompts/04-automate.md`](prompts/04-automate.md) | 日次自動化ワークフロー |

```
IG分析 → ニッチ調査 → スクリプト生成 → フック設計 → 日次自動化
 (00)      (01)          (02)            (03)         (04)
```

## 📱 マルチプラットフォーム配信

同一コンテンツを各縦型ショート動画プラットフォームへ最適化して配信します。

- **Instagram Reels**
- **TikTok**
- **YouTube Shorts**
- **LINE VOOM**（対応可能なら）

各プラットフォームの尺・アスペクト比（9:16）・キャプション・ハッシュタグ・投稿時間を最適化します。

## 🐱🐶 キャラクター

看板バディ「ネコネコ（猫）＆イヌイヌ（犬）」の詳細設定は [`docs/character.md`](docs/character.md) を参照してください。

- **ネコネコ（猫）**：のんびり博識な**ツッコミ／解説役**
- **イヌイヌ（犬）**：元気で素直な**ボケ／共感役**
- 掛け合い：犬が「あるある」でボケる→猫が根拠を添えてやさしく解説
- テーマ例：熱中症対策、栄養管理、応急処置、季節の注意点、ワクチン、デンタルケア

## ⚠️ エビデンスに関する注意

本エンジンが生成するコンテンツは獣医学的根拠に基づくことを前提とします。すべての健康・医療情報は信頼できる一次情報（PMC / ACVIM / Merck Veterinary Manual 等）に紐づけ、診断・治療の代替ではなく「早めの受診」を促すことを原則とします。詳細は [`docs/character.md`](docs/character.md) のエビデンスソースを参照してください。
